Sed CheatSheet

9月 18, 2016

  1. sed "s/my/lion's/g" exp.txt

    单引号无法用\’转义，而双引号可以使用\”进行转义

  2. sed -i "s/my/lion's/g" exp.txt

    -i 原地修改

  3. { codeblock lang:bash }
    sed ‘s/^/#/g’ exp.txt

    sed 's/$/--/g' exp.txt

    { endcodeblock }

    ^, $均为正则语法，\< 表示词首， >表示词尾。如\<abc表示以abc为首的词

    正则举例：

    { codeblock lang:bash }
    sed ‘s/<.>//g’ html.txt (x)
    sed ‘s/<[^>]\    >//g html.txt (v)
    { endblock }

  4. { codeblock lang:bash }
    sed ‘3s/my/your/g’ exp.txt
    sed ‘3,6s/my/your/g’ exp.txt
    { endblock }
    指定行替换

  5. { codeblock lang:bash }
    sed ‘s/my/your/1’ exp.txt
    sed ‘s/my/your/2’ exp.txt
    sed ‘s/my/your/3g’ exp.txt
    { endblock }

    指定行内位置替换
