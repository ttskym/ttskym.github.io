Windows 口令猜解(445，3389)

8月 24, 2015

口令猜解是远程攻击Windows系统的重要方式之一。比较常用的是利用文件和打印共享服务以及终端服务/远程桌面服务两种服务进行猜解。众多的自动化猜解工具中，以medusa和hydra表现较好，两者都包含smb模块，利用smb协议猜解，此外hydra还支持rdp协议的猜解。
